# Bank Account Simulator
A C# Console App that demonstrates core programming concepts such as object-oriented design, encapsulation, validation, and basic financial operations.
This project simulates real-world banking features like deposits, withdrawals, balance checks, and transaction history.

## Features
- Create and manage a bank account
- Deposit and withdraw money
- Validate operations
- Display current balance
- View transaction history
- Clean and modular object-oriented design
- Authentication (PIN)

## Tech Stack
- Language: c#
- Framework: .Net 8
- Paradigms: Object-Oriented Programming (OOP)

## How to run
BankAccount 
- Holds account information
BankAccountManagement
- Handles user interactions
- Uses BankAccount methods
- Loops through menu options
